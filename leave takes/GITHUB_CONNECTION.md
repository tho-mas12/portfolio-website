## 📋 Project Features (For GitHub README)

### ✨ Key Features:
- 👤 **Student & Admin Authentication** - Secure login system
- 📝 **Leave Application Form** - Apply for hostel leave with date validation
- 📊 **Admin Dashboard** - Manage and approve/reject leave requests
- 📈 **Statistics & Analytics** - View leave data with date-wise filtering
- 🗄️ **Database Viewer** - Web interface to view all database records
- 🎨 **Professional UI** - Animated, modern, responsive design
- 📱 **Mobile Friendly** - Works on all screen sizes

### 🛠️ Tech Stack:
- **Backend**: Python 3.7 + Flask
- **Database**: SQLite3
- **Frontend**: HTML5 + CSS3 (with animations) + JavaScript
- **Design**: Modern gradient UI with smooth animations

### 📊 Database Tables:
1. **User** - Stores students and admin credentials
2. **LeaveApplication** - Stores leave requests with status
